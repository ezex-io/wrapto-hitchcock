# Hitchcock (wrapto-hitchcock)

![Alfred Hitchcock portrait](hitchcock.jpg)

> ⚠️ Hitchcock is a testing toolkit. Do not use the generated keys or accounts in production.

Hitchcock is an interactive toolkit that allows Wrapto engineers to test Wrapto and identify issues before they reach production.

## Quick Start

### Install dependencies**

```bash
pip install pactus-sdk
pip install eth-account
```
### Start Hitchcock

```
python3 hitchcock.py
```

## Controls
- `Ctrl+C`: Abort the current action and return to the main menu.
- `Ctrl+D`: Exit Hitchcock immediately.
- `Enter`: Advance through menus or continue after results are displayed.
